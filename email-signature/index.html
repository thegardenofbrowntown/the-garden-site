Apple Mail instructions
  1. Click Copy HTML Code above
  2. Open Apple Mail → Settings → Signatures
  3. Add a new signature, type anything as a placeholder
  4. Quit Mail completely
  5. In Finder, go to ~/Library/Mail/V10/MailData/Signatures/
  6. Find the .mailsignature file for your new signature and open it in a text editor
  7. Replace everything after the last blank line with the copied HTML, then save
  8. Reopen Mail — your signature will appear
Outlook instructions
  1. Click Select Signature above
  2. Press ⌘C (Mac) or Ctrl+C (Windows) to copy
  3. Open Outlook → Preferences → Signatures → click +
  4. Paste with ⌘V / Ctrl+V in the signature editor
  5. Name the signature and close — done
Gmail instructions
  1. Click Select Signature above
  2. Press ⌘C to copy
  3. Open Gmail → Settings (gear icon) → See all settings → Signature
  4. Click Create new, give it a name
  5. Paste with ⌘V in the signature editor
  6. Scroll down and click Save Changes
